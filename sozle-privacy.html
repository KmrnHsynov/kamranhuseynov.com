MƏLUMAT

Hansı məlumatlar toplanır?

SÖZLƏ isteğe bağlı hesab sisteminə malikdir. Hesab yaratmadan da oynaya bilərsiniz. Əgər hesab yaratsanız, aşağıdakı məlumatlar toplanır:

  • İstifadəçi adı — siz özünüz seçirsiniz (real ad tələb olunmur)
  • Şifrə — şifrələnmiş formada saxlanılır, heç vaxt açıq mətnlə görünmür
  • Avatar seçimi — oğlan və ya qız ikonu
  • Oyun statistikası — oynanan oyunların sayı, qazanma faizi, ardıcıl günlər (streak)

Biz heç vaxt sizin real adınızı, e-poçtunuzu, telefon nömrənizi, yerləşmənizi və ya digər şəxsi məlumatlarınızı toplamırıq.

Üçüncü tərəf analitika və ya reklam xidmətlərindən istifadə edilmir.

SAXLAMA

Məlumatlar harada saxlanılır?

Hesab məlumatları Supabase (ABŞ əsaslı, SOC 2 sertifikatlı) buludunda saxlanılır. Supabase, məlumatları şifrələyərək qoruyur.

Oyun vəziyyəti (cari gün cəhdləriniz) cihazınızın lokal yaddaşında (localStorage) saxlanılır və heç vaxt xarici serverə göndərilmir.

Hesabınızı silmək istəsəniz, bizimlə əlaqə saxlayın — məlumatlarınızı tamamilə silirik.

Last updated: June 29, 2026

DATA

What data do we collect?

SÖZLƏ has an optional account system. You can play without creating an account. If you choose to create an account, the following data is collected:

  • Username — chosen by you (no real name required)
  • Password — stored in encrypted form, never in plain text
  • Avatar choice — a boy or girl icon you select
  • Game statistics — games played, win percentage, and daily streak count

We never collect your real name, email address, phone number, location, or any other personal information.

We do not use third-party analytics or advertising services.

STORAGE

Where is data stored?

Account data is stored on Supabase (a US-based, SOC 2 certified cloud provider). All data is encrypted at rest and in transit.

Your current game state (today's guesses) is stored locally on your device using browser/app storage and is never sent to any server.

To delete your account and all associated data, contact us at the email below.

SECURITY

Data security

  • Passwords are hashed using bcrypt — never stored in plain text
  • All data transmission uses HTTPS encryption
  • No email address is collected — login uses an anonymized identifier derived from your username
CHILDREN

Children's privacy

SÖZLƏ is suitable for all ages. Children can play the full game without creating an account. If an account is created, no real name or contact information is required.
